## Compile

```bash
mkdir build && cd build && cmake .. && make
```

## Run

* `rotate`: Press `Space` to rotate the shape
* `transform`: Press LMB and 1~6 to select and move the circles; Press RMB and 1~6 to rotate
* `planet`: Just animation
